# 🧠 Text Summarization Tool (CodTech Internship – Task 1)

This is a simple **Text Summarization Tool** built using Python and NLP.  
It takes a long article as input and returns a short, meaningful summary using the **Sumy** library (extractive summarization).

---

## 📌 Internship Task Objective

> **Task 1:** Create a tool that summarizes lengthy articles using Natural Language Processing techniques.  
> **Goal:** Build a Python script that shows both input and summarized output.

---

## 🛠 Technologies Used

- Python 3
- NLTK
- Sumy (LSA Summarizer)

---

## 📁 File Included

| File Name        | Description                            |
|------------------|----------------------------------------|
| `text_summary.py` | Main Python script for summarization  |
| `README.md`       | This file – project documentation     |

---

## ▶️ How to Run

1. Make sure Python is installed
2. Open terminal in the project folder
3. Install dependencies:
   ```
   bash
   pip install nltk sumy
   ```

4. Run the script:
```
   bash
   python text_summary.py
   
```
---

## 📄 Sample Output

**Extractive Summary:**

 * Global cooperation is needed for a sustainable future.
